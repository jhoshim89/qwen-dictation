# Contributing

Qwen Dictation is currently a developer MVP. The most useful contributions are
clear bug reports, install feedback, real microphone accuracy notes, and focused
pull requests.

## Useful bug reports

Please include:

- macOS version.
- Mac model and chip.
- Install method.
- Selected ASR engine.
- What app/text field you dictated into.
- Steps to reproduce.
- Relevant terminal output or logs.

## Pull requests

- Keep changes focused.
- Include tests for behavior changes when practical.
- Do not reformat unrelated files.
- Do not replace the ASR/model architecture without opening an issue first.

## Current priorities

- Install reliability.
- macOS permission onboarding.
- Local model management.
- Real microphone accuracy evaluation.
- Signed macOS app packaging.
