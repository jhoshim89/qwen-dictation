# Promo Kit

This folder keeps launch and promotion work separate from product code.

## Files

- `positioning.md`: audience, core promise, hooks, and claims to avoid.
- `launch-plan.md`: launch sequence and success metrics.
- `launch-checklist.md`: pre-launch, launch-day, and follow-up checklist.
- `channel-playbook.md`: how to post on X, Threads, Show HN, and Reddit.
- `posts.yaml`: reusable post drafts.
- `reply-bank.md`: reusable manual reply candidates.
- `metrics.md`: manual campaign tracking.
- `github-metadata.md`: repository description/topics.
- `automation-scope.md`: safe automation boundary for promotion work.

## Operating rule

Automation may draft, format, schedule, and measure posts. Replies to strangers,
DMs, likes, follows, and stars stay manual.
